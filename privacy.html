Política de Privacidade

Aplicativo: Missão Gloriosa

Última atualização: 2026

Este aplicativo é uma ferramenta de uso pessoal/interno, criada e operada pelo próprio criador de conteúdo do canal "Missão Gloriosa", para automatizar a publicação de vídeos curtos (cortes de análises e notícias) na própria conta do TikTok do criador.

1. Quais dados são acessados

Ao autorizar este aplicativo via login do TikTok, o app acessa:

  • Informações básicas do perfil público da conta (escopo user.info.basic), apenas para confirmar que a conta correta está conectada;
  • Permissão para publicar vídeos na própria conta (escopos video.publish e video.upload).

2. Como os dados são armazenados

O token de acesso (access token / refresh token) obtido após a autorização é armazenado localmente, no computador do criador, e não é transmitido, vendido ou compartilhado com terceiros.

3. Conteúdo publicado

Os vídeos publicados são cortes verticais gerados a partir de conteúdo de vídeos públicos (análises e notícias), produzidos e revisados pelo próprio criador antes da publicação.

4. Compartilhamento com terceiros

Nenhum dado obtido por meio da API do TikTok é compartilhado com terceiros. O aplicativo não é distribuído para outros usuários — é uma ferramenta de automação pessoal.

5. Revogação de acesso

O criador pode revogar o acesso do aplicativo à sua conta TikTok a qualquer momento, diretamente nas configurações de aplicativos conectados da conta TikTok.
